# install

Installation scripts for Linux, macOS, and Windows

```bash
install/
├── unix/          # Common scripts for macOS/Linux (POSIX/zsh) 
├── macos/         # Specific scripts for macOS (Homebrew, darwin)  
├── linux/         # Specific scripts for Ubuntu/Debian (apt)  
└── windows/       # Specific scripts for Windows (PowerShell)
```
